# Healthcare-asset-tracker-and-uptime-dashboard
A Python-based engineering dashboard for tracking clinical asset reliability, MTBF/MTTR analytics, and fleet uptime optimization.
The Healthcare Asset & Uptime Dashboard is a specialized engineering tool designed to bridge the gap between traditional clinical maintenance and modern reliability data science. Engineered to solve the challenge of equipment downtime in high-pressure medical environments, this project provides a robust framework for calculating and visualizing Mean Time Between Failure (MTBF) and Mean Time To Repair (MTTR) across diverse clinical fleets. By transforming raw maintenance logs—modeled after real-world internships and clinical experiences into a dynamic Predictive Maintenance interface, the dashboard allows Biomedical Engineers to identify high-failure assets and manufacturer performance gaps before they impact patient care. Developed using Python, Streamlit, and Plotly, the repository features a multi-layered analytical approach including a Reliability Matrix, geospatial mapping for regional asset distribution, and a Vendor Accountability Scorecard. This project serves as a technical proof-of-concept for how the integration of data analytics can optimize medical equipment uptime and streamline resource allocation in clinical engineering departments, demonstrating a sophisticated ability to turn technical logs into high-level operational intelligence.
